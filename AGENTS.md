# AGENTS.md - claude-code-playbook

## 핵심 원칙

1. **단일 파일 원칙**: index.html 하나로 전체 플레이북 제공
2. **크로스 플랫폼**: Mac/Windows 모두 지원하는 범용 가이드
3. **민감 정보 제외**: 토큰, API 키 등은 플레이스홀더 처리
4. **자기 완결적**: 외부 빌드 도구 없이 즉시 배포 가능
5. **프롬프트 우선**: 비개발자도 Claude Code 프롬프트로 셋업 가능
