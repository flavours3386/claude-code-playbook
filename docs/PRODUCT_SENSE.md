# Product Sense - claude-code-playbook

## 제품 방향

팀 내 Claude Code 도입을 가속화하는 셀프서비스 온보딩 가이드.
신규 팀원이 30분 내에 동일한 개발 환경을 구축할 수 있도록 한다.

## 대상 사용자

- 세일즈팀 팀원 (개발 경험 다양)
- Mac 또는 Windows 사용자
- Claude Code 처음 사용하는 사람

## 핵심 가치

1. **재현성**: 누가 따라해도 동일한 결과
2. **자기 완결성**: 외부 문서 참조 없이 이 페이지만으로 충분
3. **크로스 플랫폼**: OS 무관하게 동작

## 성공 지표

- 신규 팀원 온보딩 시간 단축
- 셋업 관련 질문/이슈 감소
- 팀 내 Claude Code 활용률 증가
